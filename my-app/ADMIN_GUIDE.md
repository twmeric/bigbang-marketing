# Big Bang Marketing 案例管理後台使用指南

## 🚀 訪問後台

後台網址：`https://bigbang.jkdcoding.com/admin`

默認密碼：`bigbang2025`

## 📋 功能說明

### 1. 添加案例
- 點擊「添加案例」按鈕
- 填寫案例信息（標題、客戶、類別、圖片路徑、描述）
- 點擊「保存」

### 2. 編輯案例
- 在案例列表中找到要編輯的案例
- 點擊右側的「編輯」圖標（鉛筆）
- 修改信息後點擊「保存」

### 3. 刪除案例
- 在案例列表中找到要刪除的案例
- 點擊右側的「刪除」圖標（垃圾桶）
- 確認刪除

### 4. 導出數據
- 點擊「導出 JSON」按鈕
- 下載 cases.json 文件
- 將文件交給開發人員更新到網站

### 5. 導入數據
- 點擊「導入 JSON」按鈕
- 選擇之前導出的 cases.json 文件
- 數據將自動加載

## 🖼️ 圖片上傳說明

由於這是靜態網站，圖片需要手動上傳：

1. 將圖片文件發送給開發人員
2. 開發人員將圖片上傳到 `public/` 文件夾
3. 在後台輸入圖片路徑，例如：
   - `/case-new.png`
   - `/images/case-1.jpg`

## 💾 數據存儲

- 數據暫時存儲在瀏覽器的 localStorage 中
- 清除瀏覽器數據會丟失未導出的更改
- **重要**：完成編輯後請務必導出 JSON 文件並發送給開發人員

## 📞 聯繫開發人員

如需更新網站內容，請將導出的 JSON 文件發送給開發團隊。
